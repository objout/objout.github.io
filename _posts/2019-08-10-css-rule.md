# 一、书写顺序

~~~css
* {  
    
    /*显示属性*/  
    display  
    position  
    float  
    clear  
    cursor  
    …  

    /*盒模型*/  
    margin  
    padding  
    width  
    height  

    /*排版*/  
    vertical-align  
    white-space  
    text-decoration  
    text-align  
    …

    font  
    content  

    /*边框背景.把  
    boder和background放在最后,原因是修改     的频
    率会较之前的频繁,放在最后查看起来方     便。*/  
    border  
    background  
}
~~~
